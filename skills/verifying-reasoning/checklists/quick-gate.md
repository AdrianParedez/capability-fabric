# Checklist: quick gate (60 seconds, medium stakes)

The fast version. For high-stakes/irreversible, use the full pass + an objective test.

- [ ] Can I state the claim and the **exact question** it answers in one line each?
- [ ] What's the **shakiest assumption**? Is it actually true?
- [ ] One real attempt to **disconfirm**, a counterexample or an inversion. Found one?
- [ ] If a **cheap test exists**, did I run it (instead of reasoning about it)?
- [ ] Did I check at least one **edge** (empty/zero/null/error path)?
- [ ] Does the answer match **the question asked**, not a near-miss?
- [ ] Verdict stated: **confirmed / caveats / refuted / unknown** (+ next step if not confirmed).
